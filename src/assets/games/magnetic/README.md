# Magnetic Slot — Textures

Положи сюда **6 PNG-файлов** по 16×16 пикселей (image-rendering:
pixelated в CSS отмасштабирует их без блюра под `--cell-h: 38px`).

| Файл           | Что это                | Тир / роль       |
|----------------|------------------------|------------------|
| `coin.png`     | 🪙 монетка             | 25 %             |
| `bolt.png`     | 🔩 болт                | 50 %             |
| `compass.png`  | 🧭 компас              | 75 %             |
| `orb.png`      | 🔮 плазма-орб          | 100 %            |
| `gem.png`      | 💎 алмаз               | scatter          |
| `magnet.png`   | 🧲 магнит (header)     | — (только наверху) |

### Палитра слота сейчас
- Фон: фиолетовый/magenta (`#1a0a3e`, `#5223a8`)
- Акценты: violet-300 / fuchsia-300
- Tier-ячейки: полупрозрачные violet
- Спин-кнопка: fuchsia→purple неон

### Подсказки

- **Контрастность:** тёмный outline 1–2 px по краю символа улучшит читаемость на фоне.
- **Палитра:** тёплые/контрастные тона (золото, серебро, медь, циан) отлично смотрятся на фиолетовом фоне.
- **Размер:** 16×16 — sweet spot, как в Pixel Mine. Можно и 24×24 если хочется больше деталей, но 16 проще быстрее.
- **Прозрачность:** фон PNG должен быть прозрачным.

После того как файлы лежат тут — MagneticSlot.jsx импортирует их через
`import coinTex from '../assets/games/magnetic/coin.png'` и подключит
вместо emoji-полей в массиве `SYMBOLS`.
